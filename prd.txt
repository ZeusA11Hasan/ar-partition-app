## 1. Module Breakdown

### Module 1 — AR Engine
**Responsibility:** Manage WebXR session lifecycle, hit-test, frame loop, reference spaces.  
**Files:** `src/xr/session.js`, `src/xr/hitTest.js`, `src/xr/frameLoop.js`  
**Dependencies:** None (native WebXR API)

### Module 2 — Drawing Engine
**Responsibility:** Capture touch strokes on floor plane, run PCA correction, render corrected line, expose endpoint handles.  
**Files:** `src/xr/drawing.js`, `src/utils/lineCorrection.js`  
**Dependencies:** Module 1 (AR Engine), math.js

### Module 3 — Anchor System
**Responsibility:** Create XRAnchor at wall endpoints, update wall transforms each frame, host to Google Cloud Anchors, resolve from Cloud Anchors.  
**Files:** `src/xr/anchors.js`, `src/api/cloudAnchors.js`  
**Dependencies:** Module 1 (AR Engine)

### Module 4 — Storage Layer
**Responsibility:** IndexedDB CRUD for projects, layouts, worlds. Map cloud anchor IDs to wall records.  
**Files:** `src/db/storage.js`, `src/db/schema.js`  
**Dependencies:** idb library

### Module 5 — World Generation
**Responsibility:** Upload video to World Labs, trigger world generation, poll operation, store result URLs.  
**Files:** `src/api/worldLabs.js`, `src/hooks/useWorldGeneration.js`  
**Dependencies:** Module 4 (Storage Layer), axios

### Module 6 — UI Layer
**Responsibility:** All React screens, components, Zustand state, routing, PWA config, Spark world viewer.  
**Files:** `src/screens/`, `src/components/`, `src/store/useAppStore.js`  
**Dependencies:** All modules above

### 3.4 Line Correction — `src/utils/lineCorrection.js`

```typescript
filterOutliers(points: {x: number, z: number}[]): {x: number, z: number}[]
// Remove points > 2 standard deviations from centroid
// Returns filtered array

correctLine(points3D: Vector3[]): CorrectedLine | null
// Inputs: raw Vector3 array (use only X and Z — ignore Y)
// Outputs: { start: Vector3, end: Vector3, length: number, angle: number }
// Algorithm:
//   1. Map to 2D: pts = points3D.map(p => [p.x, p.z])
//   2. Call filterOutliers(pts)
//   3. Compute centroid: cx = mean(pts.x), cz = mean(pts.z)
//   4. Center points: centered = pts.map(p => [p[0]-cx, p[1]-cz])
//   5. Build 2x2 covariance matrix
//   6. Run math.svd(covMatrix) → U, S, V
//   7. First column of V = dominant direction [dx, dz]
//   8. Project all points: proj[i] = pts[i][0]*dx + pts[i][1]*dz
//   9. minP = min(proj), maxP = max(proj)
//   10. start = { x: cx + minP*dx, y: floorY, z: cz + minP*dz }
//   11. end   = { x: cx + maxP*dx, y: floorY, z: cz + maxP*dz }
//   12. length = distance(start, end)
//   13. angle = Math.atan2(dz, dx) * (180 / Math.PI)
//   14. Return { start, end, length, angle }
// Fallback (SVD fails or < 5 points): return first-to-last raw points

snapAngle(line: CorrectedLine): CorrectedLine
// Inputs: corrected line
// Outputs: same line with angle snapped if within threshold
// Snap targets: 0°, 45°, 90°, 135°, 180°, 225°, 270°, 315°
// Threshold: 8 degrees
// If snap: rotate end point around start point to exact snap angle
// Return new line object (do not mutate input)
```

### 3.5 Partition Mesh — `src/utils/partitionMesh.js`

```typescript
buildWallMesh(wall: WallRecord): THREE.Group
// Inputs: wall record from store
// Outputs: Three.js Group containing:
//   - Main slab: BoxGeometry(length, height, 0.075)
//   - Panel lines: LineSegments every 0.6m along length
//   - Material: MeshStandardMaterial based on wall.type
//     solid: { color: wall.colour, roughness: 0.8 }
//     glass: { color: 0xaaddff, transparent: true, opacity: 0.4 }
//     mesh:  { color: wall.colour, wireframe: true }
//     half:  height = wall.height / 2, same as solid
// Side effects: none
// Note: mesh is centred at origin — caller positions it

updateMeshTransform(
  mesh: THREE.Group,
  startPose: XRPose,
  endPose: XRPose
): void
// Inputs: mesh Group, start anchor pose, end anchor pose
// Side effects:
//   - Computes midpoint between start and end poses
//   - Sets mesh.position to midpoint
//   - Computes angle between start and end on XZ plane
//   - Sets mesh.rotation.y to that angle
//   - Sets mesh.scale if length changed (endpoint drag)

buildDoorMesh(panel: PanelRecord, wallHeight: number): THREE.Group
// Outputs: Group with:
//   - Frame: 4 BoxGeometry bars forming rectangle (0.05m thick)
//   - Slab: BoxGeometry(0.55, wallHeight - 0.1, 0.04) — the door panel
//   - Handle: CylinderGeometry(0.01, 0.01, 0.12) at 1.0m height, 0.05m from edge
//   - Hinge side: determined by panel.doorSide ('left' | 'right')
// Door open animation: rotate slab Group on Y axis 0→-90° in 400ms (use framer-motion spring or manual lerp)
```

## 4. API Contracts

### 4.1 World Labs API

**Base URL:** `https://api.worldlabs.ai/marble/v1`  
**Auth header:** `Authorization: Bearer ${import.meta.env.VITE_WORLD_LABS_API_KEY}`

#### POST /media-assets
```
Request:
  Content-Type: application/json
  Body: { "file_name": "ar_session.mp4", "kind": "video", "extension": "mp4" }

Response 200:
  {
    "media_asset": { "id": "uuid", "file_name": "ar_session.mp4", "kind": "video" },
    "upload_info": {
      "upload_url": "https://storage.googleapis.com/...",
      "upload_method": "PUT",
      "required_headers": { "x-goog-content-length-range": "0,1048576000" }
    }
  }

Error states:
  401 → invalid API key → throw Error('WORLD_LABS_AUTH_FAILED')
  429 → rate limited → wait 60s, retry once
  500 → server error → throw Error('WORLD_LABS_SERVER_ERROR')
```

#### PUT {upload_url}
```
Request:
  Headers: { "x-goog-content-length-range": "0,1048576000" }
  Body: raw video Blob (binary)

Response: 200 empty body on success
Error: any non-200 → retry up to 3x with 2s delay
```

#### POST /worlds:generate
```
Request:
  Content-Type: application/json
  Body: {
    "display_name": string,
    "world_prompt": {
      "type": "video",
      "video_prompt": {
        "source": "media_asset",
        "media_asset_id": string
      },
      "text_prompt": string
    }
  }

Response 200:
  { "operation_id": "uuid" }

Error states:
  400 → bad request (invalid media_asset_id) → throw Error('WORLD_LABS_BAD_REQUEST')
  402 → insufficient credits → throw Error('WORLD_LABS_QUOTA_EXCEEDED')
```

#### GET /operations/{operation_id}
```
Response (pending):
  {
    "done": false,
    "metadata": {
      "progress": { "status": "RUNNING", "description": "Processing video..." }
    }
  }

Response (complete):
  {
    "done": true,
    "error": null,
    "response": {
      "id": "world-uuid",
      "world_marble_url": "https://marble.worldlabs.ai/world/...",
      "assets": {
        "splats": { "spz_urls": { "500k": "url", "full_res": "url" } },
        "mesh": { "collider_mesh_url": "url" },
        "imagery": { "pano_url": "url" }
      }
    }
  }

Response (failed):
  { "done": true, "error": "Generation failed: insufficient visual data" }
  → throw Error with error string
```

### 4.2 Google Cloud Anchors (WebXR Session Methods)

```
session.hostCloudAnchor(xrAnchor, { daysToExpire: 365 })
  → Promise<string>  (cloudAnchorId)
  → Requires OAuth keyless auth (not API key for 365d TTL)
  → Error codes: 'insufficient_visual_data' | 'service_unavailable' | 'network_error'

  → Promise<XRAnchor>

      'pending'              → keep polling
      'success'              → resolved, use anchor
      'error_not_localized'  → user needs to move camera
      'error_not_found'      → anchor expired or invalid
      'error_cloud_service'  → Google API error

xrAnchor.cloudAnchorState
  → string, read each frame or on interval
```

### State Transitions

```
APP START:
  arSupported = false
  → checkXRSupport() → arSupported = true/false

SESSION START (Draw/Preview):
  xrSession = null → startXRSession() → xrSession = XRSession
  xrError = null

DRAWING:
  isDrawing = false
  → touchstart → isDrawing = true, currentStroke = { rawPoints: [], startTime }
  → touchend → endStroke() → pendingLine = CorrectedLine | null
  → confirm → pendingLine cleared, wall added to walls[]

WALL CONFIRMED:
  walls = [...walls, newWall]
  localAnchors.set(wall.id, { start, end })
  → hostAnchor() → wall.cloudAnchorIdStart + cloudAnchorIdEnd updated
  → saveLayout() to IndexedDB

WORLD GEN:
  worldGenStatus: idle → uploading → generating → ready | error
  worldGenProgress updated each poll

SESSION END:
  xrSession = null
  isDrawing = false
  currentStroke = null
  pendingLine = null
```

## 7. Edge Case Handling

### Anchor Failure
```
Condition: resolveAnchor returns null (timeout or error state)
Handler:
  1. Load wall.startLocal + wall.endLocal from IndexedDB
  2. Place wall mesh at stored local coordinates
  3. Render wall with dashed orange outline (visual indicator: estimated position)
  4. Show toast: "Wall position estimated — walk around to improve accuracy"
  5. Keep polling in background — if anchor resolves later, snap mesh to real position
```

### No Floor Detection
```
Condition: getHitTestResult returns null for > 5 seconds
Handler:
  1. Show overlay: "Point camera slowly at floor"
  2. Increase FloorIndicator opacity animation to draw attention
  3. If still null after 15s: show "Low texture floor detected" + tips (add mat, papers)
  4. Do not allow startStroke() until hitResult is non-null
```

### Low Light
```
Condition: ARCore tracking state degrades (xrFrame.session.visibilityState)
Handler:
  1. Monitor session.visibilityState — if 'hidden' or tracking lost: pause anchor updates
  2. Show banner: "Tracking paused — increase lighting"
  3. Resume updates when tracking restored
  Note: WebXR does not expose light level directly — use visibilityState as proxy
```

### World Labs API Failure
```
Condition: generateWorld throws or pollOperation times out
Handler:
  1. Set store.worldGenStatus = 'error'
  2. Show error message from thrown Error
  3. Show "Retry" button → re-run useWorldGeneration from uploadVideo step
  4. Preserve videoBlob in store so re-upload doesn't need re-record
  5. Show "Save video" button → let user download raw recording as fallback
```

### PCA SVD Failure
```
Condition: math.svd throws (malformed matrix, < 2 points after filter)
Handler:
  1. Catch error in correctLine()
  2. Fallback: return { start: rawPoints[0], end: rawPoints[last], length, angle }
  3. Log to store.debugErrors[]
  4. No UI error shown — fallback is good enough
```

### IndexedDB Unavailable
```
Condition: initDB() throws (private browsing, storage quota exceeded)
Handler:
  1. Set module flag: DB_AVAILABLE = false
  2. All storage calls become no-ops returning undefined/[]
  3. Show persistent banner: "Storage unavailable — data will not be saved"
  4. App still functions for current session only
```

### MediaRecorder Not Supported
```
Condition: typeof MediaRecorder === 'undefined' OR canvas.captureStream fails
Handler:
  1. Disable "Record Tour" button
  2. Show tooltip: "Recording not supported on this browser"
  3. Mode 3 falls back to: upload a video file from camera roll instead
```

### WebXR Anchors Feature Missing
```
Condition: 'anchors' not in session.enabledFeatures after session start
Handler:
  1. Set store.anchorsSupported = false
  2. Use session-only local anchors only (no Cloud Anchor hosting)
  3. Show banner: "Persistent anchors unavailable — walls reset on app restart"
  4. Hide "Live AR Preview" mode card on ProjectScreen
```

```javascript
export async function startXRSession(canvasRef) {
  const renderer = new THREE.WebGLRenderer({ canvas: canvasRef.current, antialias: true })
  renderer.xr.enabled = true

  const session = await navigator.xr.requestSession('immersive-ar', {
    requiredFeatures: ['hit-test', 'anchors'],
    optionalFeatures: ['dom-overlay'],
    domOverlay: { root: document.getElementById('ar-ui') }
  })

  renderer.xr.setReferenceSpaceType('local-floor')
  await renderer.xr.setSession(session)

  const refSpace = await session.requestReferenceSpace('local-floor')
  const hitTestSource = await initHitTest(session)

  useAppStore.getState().setXRSession(session)
  useAppStore.getState().setXRRefSpace(refSpace)

  startFrameLoop(session, renderer, refSpace, hitTestSource)
  return session
}
```

### 8.3 PCA Core
```javascript
// src/utils/lineCorrection.js
import { svd, mean } from 'mathjs'

export function correctLine(points3D) {
  if (points3D.length < 2) return null
  const pts = points3D.map(p => [p.x, p.z])
  const filtered = filterOutliers(pts)
  if (filtered.length < 2) return null

  const cx = mean(filtered.map(p => p[0]))
  const cz = mean(filtered.map(p => p[1]))
  const centered = filtered.map(p => [p[0] - cx, p[1] - cz])

  const cov = [
    [mean(centered.map(p => p[0]*p[0])), mean(centered.map(p => p[0]*p[1]))],
    [mean(centered.map(p => p[1]*p[0])), mean(centered.map(p => p[1]*p[1]))]
  ]

  try {
    const { U, S, V } = svd(cov)
    const [dx, dz] = [V[0][0], V[1][0]]

    const projs = filtered.map(p => p[0]*dx + p[1]*dz)
    const minP = Math.min(...projs)
    const maxP = Math.max(...projs)

    const floorY = points3D[0].y
    return {
      start: new THREE.Vector3(cx + minP*dx, floorY, cz + minP*dz),
      end:   new THREE.Vector3(cx + maxP*dx, floorY, cz + maxP*dz),
      length: Math.abs(maxP - minP),
      angle:  Math.atan2(dz, dx) * (180 / Math.PI)
    }
  } catch {
    const first = points3D[0], last = points3D[points3D.length - 1]
    return { start: first, end: last, length: first.distanceTo(last), angle: 0 }
  }
}
```

```javascript
// src/db/schema.js
export const DB_NAME = 'partitionAR_db'
export const DB_VERSION = 1
export const STORES = { PROJECTS: 'projects', LAYOUTS: 'layouts', WORLDS: 'worlds', SESSIONS: 'sessions' }

export function upgrade(db) {
  if (!db.objectStoreNames.contains(STORES.PROJECTS)) {
    db.createObjectStore(STORES.PROJECTS, { keyPath: 'id' })
  }
  if (!db.objectStoreNames.contains(STORES.LAYOUTS)) {
    const ls = db.createObjectStore(STORES.LAYOUTS, { keyPath: 'id' })
    ls.createIndex('projectId', 'projectId', { unique: false })
  }
  if (!db.objectStoreNames.contains(STORES.WORLDS)) {
    const ws = db.createObjectStore(STORES.WORLDS, { keyPath: 'id' })
    ws.createIndex('layoutId', 'layoutId', { unique: false })
  }
  if (!db.objectStoreNames.contains(STORES.SESSIONS)) {
    db.createObjectStore(STORES.SESSIONS, { keyPath: 'id' })
  }
}
```

```javascript
// src/api/worldLabs.js (pollOperation)
export async function pollOperation(operationId, onProgress) {
  const MAX_POLLS = 90 // 15 minutes at 10s interval
  let polls = 0

  while (polls < MAX_POLLS) {
    await new Promise(r => setTimeout(r, 10000))
    polls++

    const res = await axios.get(
      `${BASE_URL}/operations/${operationId}`,
      { headers: { Authorization: `Bearer ${API_KEY}` } }
    )
    const op = res.data

    if (op.metadata?.progress) {
      onProgress(op.metadata.progress.status, op.metadata.progress.description)
    }

    if (op.done) {
      if (op.error) throw new Error(op.error)
      const r = op.response
      return {
        worldId: r.id,
        splatUrl500k: r.assets.splats.spz_urls['500k'],
        splatUrlFullRes: r.assets.splats.spz_urls['full_res'],
        colliderMeshUrl: r.assets.mesh.collider_mesh_url,
        panoramaUrl: r.assets.imagery.pano_url,
        worldMarbleUrl: r.world_marble_url
      }
    }
  }

  throw new Error('timeout')
}
```
